# ПЭТ-Хаус НН

## Структура проекта

```
pet-housenn/
├── dist/             # Собранные файлы для продакшена
├── public/           # Статические файлы
│   └── images/
├── src/              # Исходные файлы
│   ├── components/   # HTML компоненты
│   ├── js/           # JavaScript файлы
│   ├── pages/        # Страницы сайта
│   └── styles/       # SCSS стили
│       ├── scss/     # Исходные SCSS файлы
│       └── css/      # Скомпилированные CSS файлы
├── index.html        # Главная страница
├── vite.config.js    # Конфигурация
└── package.json
```

## Установка

```bash
npm install
```

## Команды

### Разработка
```bash
npm run dev
```

### Сборка
```bash
npm run build
```

### Просмотр собранной версии
```bash
npm run preview
```
